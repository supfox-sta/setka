# <img align="right" src="/Web/static/img/logo_shadow.png" alt="Сетка" title="Сетка" width="15%">Сетка | Социальная сеть ностальгии

_[English](README.md)_

**Сетка** — это бесплатная и открытая платформа социальной сети, вдохновленная классическим интерфейсом и функциями ВКонтакте (VK). Она создана для любителей ностальгии, которые скучают по опыту общения в социальных сетях начала 2000-х.

Этот проект основан на [OpenVK](https://github.com/openvk/openvk), фанатской инициативе, не связанной с ВКонтакте или его материнской компанией VK Ltd. Он построен с использованием PHP и современных веб-технологий для воссоздания знакомого ощущения ранних социальных медиа.

**Примечание для пользователей GitHub**: Этот репозиторий был очищен для публичного распространения. Конфигурационные файлы (`openvk.yml`) имели конфиденциальную информацию (API ключи, пароли, личные данные), замененную на плейсхолдеры. Вам нужно настроить их перед развертыванием.

## Возможности

### Основные социальные функции
- **Профили пользователей**: Создавайте и настраивайте личные профили с фотографиями, статусами и личной информацией
- **Посты и стены**: Делитесь текстовыми постами, фото и ссылками на своей стене или стенах друзей
- **Система друзей**: Общайтесь с другими пользователями, управляйте запросами в друзья и отношениями
- **Группы и сообщества**: Создавайте и присоединяйтесь к группам по интересам, участвуйте в обсуждениях
- **Личные сообщения**: Отправляйте личные сообщения другим пользователям с чатом в реальном времени

### Медиа и контент
- **Фотоальбомы**: Загружайте и организуйте фото в альбомы с описаниями
- **Видеохостинг**: Загружайте и делитесь видео с сообществом
- **Музыкальная библиотека**: Слушайте и делитесь музыкальными треками
- **Заметки**: Пишите и публикуйте более длинный контент в виде заметок
- **Опросы**: Создавайте опросы для вовлечения аудитории

### Продвинутые функции
- **Лента новостей**: Следите за обновлениями от друзей и групп
- **Уведомления**: Получайте уведомления о важных событиях
- **Поиск**: Находите пользователей, группы и контент по всей сети
- **Поддержка API**: Интегрируйтесь с внешними приложениями через REST API
- **Темы и кастомизация**: Выбирайте из различных тем и настраивайте свой опыт
- **Многоязычная поддержка**: Доступно на нескольких языках, включая русский, английский и другие

### Администрирование и модерация
- **Панель администратора**: Комплексные инструменты для управления сетью
- **Модерация пользователей**: Инструменты для модерации контента и пользователей
- **Система поддержки**: Встроенная система тикетов для поддержки пользователей
- **Ограничение скорости**: Предотвращайте спам и злоупотребления с настраиваемыми лимитами
- **Резервное копирование**: Инструменты для резервного копирования и восстановления базы данных

### Технические возможности
- **Масштабируемая архитектура**: Построено на Chandler Application Server для высокой производительности
- **Поддержка баз данных**: MySQL/MariaDB с опциональным ClickHouse для аналитики
- **Безопасность**: Встроенные функции безопасности, включая защиту от CSRF, санитизацию ввода
- **Адаптивность**: Отлично работает на десктопе и мобильных устройствах
- **Поддержка Docker**: Легкое развертывание с Docker контейнерами
- **Готовность к Kubernetes**: Конфигурации развертывания для production

## Могу ли я создать свою собственную инстанцию Сетка?

Да! И всегда пожалуйста.

Однако, Сетка использует Chandler Application Server. Это программное обеспечение требует расширений, которые могут быть не предоставлены вашим хостинг-провайдером (а именно, sodium и yaml. Эти расширения доступны на большинстве хостингов ISPManager).

Если хотите, вы можете добавить вашу инстанцию в список выше, чтобы люди могли зарегистрироваться там.

### Если мой сайт использует OpenVK, должен ли я публиковать его исходные тексты?

Это зависит от обстоятельств. Вы можете оставить исходные тексты при себе, если не планируете распространять бинарники вашего сайта. Если программное обеспечение вашего сайта должно распространяться, оно может оставаться не-OSS при условии, что OpenVK не используется в качестве основного приложения и не модифицируется. Если вы модифицировали OpenVK для своих нужд или ваша работа основана на нем и вы планируете ее распространять, то вы должны лицензировать ее на условиях любой совместимой с LGPL лицензии (например, OSL, GPL, LGPL и т.д.).
